Plataforma educacional focada na comunidade brasileira de Path of Exile e Path of Exile 2.

## Objetivos

- Guias para iniciantes
- Explicações de mecânicas
- Progressão de personagens
- Sistemas de crafting
- Conteúdo educacional

## Tecnologias

- React
- Vite
- React Router
- GitHub Pages
